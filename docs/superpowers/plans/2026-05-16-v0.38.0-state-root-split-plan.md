# v0.38.0 State-Root Split Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Split the v0.31.0 `cw_state_root` umbrella into two helpers: `cw_global_state_root` (per-machine config — medic, contracts.yaml, commanders.yaml, providers-*.txt, archive/) vs `cw_state_root` (per-project state — state/, _args/, _run/, untouched). Migrate 12+ per-machine call sites that v0.31.0 made project-local. Drop `commands/medic.md` Step 1 args-file boilerplate (medic takes no args). Add permanent lint enforcing discipline.

**Architecture:** Two-helper system. `cw_global_state_root` always = `${CLONE_WARS_HOME:-$HOME/.clone-wars}` regardless of `$PWD`. `cw_state_root` unchanged (still per-project). Authors pick by data ownership: who needs this — the user's install (global) or this project (project-local)?

**Tech Stack:** Pure bash 4.2+ + tmux 3.0+ Claude Code plugin. `tests/run.sh`; `tests/lib/assert.sh`; static-wiring locks version-stamped. v0.37.0 introduced permanent lint pattern (no skip-guard); v0.38.0's `test_state_root_discipline.sh` follows the same pattern.

**Spec:** `docs/superpowers/specs/2026-05-16-v0.38.0-state-root-split-design.md` (committed `18d13f1`)

**Branch base:** `feat/v0.38.0-state-root-split` off main (post-v0.37.0 merge at `65fb96a`).

**TDD ordering:** T1 writes the new helpers with unit tests. T2 writes the permanent lint as an intentional RED baseline (catches all 12+ existing sites). T3 (lib), T4 (bin), T5 (commands) progressively eliminate violations until lint passes GREEN. T6 locks invariants + bumps versions. T7 final suite + PR.

```bash
# cw_global_state_root — per-MACHINE config root.
# Always resolves to ${CLONE_WARS_HOME:-$HOME/.clone-wars}, regardless of $PWD.
# Use for config that is per-install, not per-project: medic's
# providers-{active,available}.txt, contracts.yaml, commanders.yaml,
# the archive/ subtree.
#
# Contrast with cw_state_root (per-PROJECT state at $PWD/.clone-wars/)
# for /consult, /deploy, /deep-research, /meditate per-topic work.
#
# v0.38.0: introduced to fix the medic→consult break where v0.31.0's
# project-local default trapped per-machine config under
# <project>/.clone-wars/, but various scripts kept reading from
# ~/.clone-wars/ via literal $HOME paths. See
# docs/superpowers/specs/2026-05-16-v0.38.0-state-root-split-design.md.
cw_global_state_root() {
  printf '%s\n' "${CLONE_WARS_HOME:-$HOME/.clone-wars}"
}

# cw_global_state_ensure — like cw_state_ensure but for the global root.
# Auto-creates .gitignore = '*' on first call. ~/.clone-wars/ is not
# usually under git, but the defensive write costs nothing and covers
# the case where a user nests CLONE_WARS_HOME inside an unrelated repo.
cw_global_state_ensure() {
  local root; root=$(cw_global_state_root)
  mkdir -p "$root"
  [[ -f "$root/.gitignore" ]] || printf '*\n' > "$root/.gitignore" 2>/dev/null || true
}
```

```bash
#!/usr/bin/env bash
# tests/test_state_root_discipline.sh

# per-project state must use cw_state_root. v0.31.0 collapsed both
# under cw_state_root; v0.38.0 split them via cw_global_state_root.
# No version skip-guard.
set -euo pipefail
cd "$(dirname "$0")"
source lib/assert.sh
PLUGIN_ROOT="$(cd .. && pwd)"

FAILED=0
report() {
  echo "FAIL: $1" >&2; echo "$2" >&2; FAILED=1

# Assert 1: medic.{md,sh} never call cw_state_root (medic is per-MACHINE)
for f in commands/medic.md bin/medic.sh; do
  hits=$(grep -nE '\bcw_state_root\b' "$PLUGIN_ROOT/$f" 2>/dev/null || true)
  [[ -z "$hits" ]] || report "$f calls cw_state_root (medic is per-machine; use cw_global_state_root)" "$hits"
done

# Assert 2: lib/contracts.sh + lib/commanders.sh + cw_active_providers_path
# use cw_global_state_root (the data they touch is per-machine)
for f in lib/contracts.sh lib/commanders.sh; do
  grep -qE 'cw_global_state_root' "$PLUGIN_ROOT/$f" \
    || report "$f missing cw_global_state_root call" "(expected at least one reference)"
  hits=$(grep -nE '\bcw_state_root\b' "$PLUGIN_ROOT/$f" 2>/dev/null || true)
  [[ -z "$hits" ]] || report "$f calls cw_state_root (its config is per-machine)" "$hits"
done
grep -qE 'cw_active_providers_path.*cw_global_state_root|cw_global_state_root.*cw_active_providers_path' \
  "$PLUGIN_ROOT/lib/state.sh" \
  || report "lib/state.sh cw_active_providers_path must use cw_global_state_root" "(not found)"

# Assert 3: archive base files use cw_global_state_root
for f in bin/consult-archive.sh bin/deploy-archive.sh lib/ipc.sh; do
  grep -qE 'cw_global_state_root' "$PLUGIN_ROOT/$f" \
    || report "$f archive base must use cw_global_state_root" "(expected at least one reference)"
done

# Assert 4: no literal ${CLONE_WARS_HOME:-$HOME/.clone-wars} outside lib/state.sh
# (use cw_global_state_root instead)
banned=$(grep -rnE '\$\{CLONE_WARS_HOME:-\$HOME/\.clone-wars\}' \
  "$PLUGIN_ROOT/commands" "$PLUGIN_ROOT/bin" "$PLUGIN_ROOT/lib" "$PLUGIN_ROOT/hooks" \
  2>/dev/null | grep -vE 'lib/state\.sh' || true)
[[ -z "$banned" ]] || report "literal \${CLONE_WARS_HOME:-\$HOME/.clone-wars} found outside lib/state.sh" "$banned"

hits=$(grep -nE '_args/' "$PLUGIN_ROOT/commands/medic.md" 2>/dev/null || true)
[[ -z "$hits" ]] || report "commands/medic.md references _args/ (Step 1 boilerplate should be dropped)" "$hits"

if (( FAILED != 0 )); then
  echo "" >&2
  echo "Fix: use cw_global_state_root for per-machine config (~/.clone-wars/)" >&2
  echo "     use cw_state_root for per-project state (\$PWD/.clone-wars/)" >&2
  exit 1
fi
pass "state-root discipline enforced: medic + contracts + commanders + archive use cw_global_state_root; no literal env-var seam outside lib/state.sh"
echo "test_state_root_discipline: lint passed"
```

```bash
# Version-stamped invariant lock for v0.38.0.

if grep -qE '\bcw_state_root\b' "$PLUGIN_ROOT/bin/medic.sh"; then

for f in lib/contracts.sh lib/commanders.sh; do
  grep -q 'cw_global_state_root' "$PLUGIN_ROOT/$f" \

pass "5. lib/contracts.sh + lib/commanders.sh use cw_global_state_root"

for f in bin/consult-archive.sh bin/deploy-archive.sh lib/ipc.sh; do
```
