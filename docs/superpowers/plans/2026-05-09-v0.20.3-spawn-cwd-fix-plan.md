# v0.20.3 Sub-Repo Trooper Spawn cwd Discipline — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Apply four coordinated layers so multi-repo `/clone-wars:deploy` preflight panes are born already-rooted in their sub-repo cwd via `tmux split-window -c` (no inherited hub-cwd, no `cd` later), and the respawn into the trooper TUI uses native `tmux respawn-pane -c` instead of the cd-then-exec hack.

**Architecture:** Bottom-up apply order — helper change first, then per-repo file write, then preflight flag, then directive thread, then test + version bump. Each layer is independently revertible. Single-repo `/clone-wars:deploy` and all of `/clone-wars:consult` byte-equal v0.20.2.

**Tech Stack:** pure bash + tmux 3.4+ + file IPC. No Node/Python runtime. Test framework: `tests/run.sh` runs every `tests/test_*.sh`; helpers live at `tests/lib/assert.sh`.

**Spec:** `docs/superpowers/specs/2026-05-09-v0.20.3-spawn-cwd-fix-design.md` (committed `152b2f9`).

**Branch:** `fix/v0.20.3-spawn-cwd-fix` (already created off `main` at `b0bf1a3`; spec already committed).

for i in "${!ROSTER[@]}"; do
  IFS=$'\t' read -r prov cmdr <<<"${ROSTER[$i]}"
  sentinel=$(build_sentinel "$cmdr" "$prov" "$TOPIC")
  # v0.20.3: per-pane -c when CMDR_TO_CWD has an entry; empty array = byte-equal.
  SPLIT_C_FLAG=()
  if [[ -n "${CMDR_TO_CWD[$cmdr]:-}" ]]; then

  fi
  PANE=$(tmux split-window -P -F '#{pane_id}' "$SPLIT_FLAG" -t "$PREV_PANE" "${SPLIT_C_FLAG[@]}" "$sentinel") || {

  cw_pane_label_set "$PANE" "$cmdr" "$prov" "$TOPIC" || {

    exit 1
  }
  printf '%s\t%s\n' "$cmdr" "$PANE" >> "$TMP_FILE"

  SPLIT_FLAG="-v"
done

# Verify each pane's pane_current_path matches the assigned cwd

  pane="${line#*$'\t'}"
  expected_cwd=""
  case "$cmdr" in
    rex)  expected_cwd="$DIR_A" ;;
    cody) expected_cwd="$DIR_B" ;;
    *)    echo "FAIL: unexpected cmdr '$cmdr'" >&2; exit 1 ;;
  esac
  actual_cwd=$(tmux display-message -p -t "$pane" '#{pane_current_path}')
  [[ "$actual_cwd" == "$expected_cwd" ]] \
    || { echo "FAIL: pane $pane ($cmdr) cwd '$actual_cwd' != '$expected_cwd'" >&2; exit 1; }
done
