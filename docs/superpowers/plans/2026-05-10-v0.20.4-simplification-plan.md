## Task 2: Fix `bin/consult-drilldown.sh:113` collision-counter regex over-strip

**Files:**
- Modify: `bin/consult-drilldown.sh:107-118` (the `resolve_out_path` inner function)

**The bug:** `base="${base%-[0-9]*}"` is a bash glob shortest-match suffix removal. Pattern `-[0-9]*` matches `-<digit><any-trailing-chars>`. For input `drilldown-mysection-rex` it strips `-mysection-rex` (because `m` doesn't have a leading `-` so the match anchors at the LAST `-`, which is `-rex`... actually let me re-trace) — for `drilldown-mysection-rex-2` the loop intends to strip just `-2` to re-append a fresh counter. But for fresh input `drilldown-mysection-rex`, the strip would attempt `-[0-9]*` which has NO digit-led suffix, so the strip is a no-op (good). The bug surfaces on COLLISION: when `drilldown-foo-2025-arch-rex.md` exists, the code strips `-2025-arch-rex` (because shortest suffix matching `-[0-9]*` is `-2025-arch-rex`), losing the section name entirely.

**The fix:** anchor the strip to a digits-only trailing counter via regex match.

- [ ] **Step 1: Apply the fix**

In `bin/consult-drilldown.sh`, replace line 113:

```bash
    base="${base%-[0-9]*}"
```

with:

```bash
    if [[ "$base" =~ ^(.+)-[0-9]+$ ]]; then base="${BASH_REMATCH[1]}"; fi
```

- [ ] **Step 2: Run the full test suite**

```bash
cd /home/liupan/CC/clone-wars && bash tests/run.sh
```

Expected: green. Targeted regression test lands in Task 12.

- [ ] **Step 3: Commit**

```bash
git add bin/consult-drilldown.sh
git commit -m "$(cat <<'EOF'
fix(consult-drilldown): anchor collision counter strip to digits-only suffix

Today: ${base%-[0-9]*} is a bash glob shortest-match strip. Pattern
matches `-<digit><anything>`, so for drilldown-foo-2025-arch-rex it
strips `-2025-arch-rex`, losing the section name entirely on collision.

Fix: use regex match anchored to a trailing digits-only counter.
Re-runs (drilldown-foo-rex → drilldown-foo-rex-2 → drilldown-foo-rex-3)
```

```bash
# (mirrors the latent issue v0.20.3 fixed where pane cwd inheritance
# was silently wrong).
declare -A CMDR_TO_CWD=()
if [[ -n "$CWD_FROM" ]]; then

    cwd_lineno=$((cwd_lineno + 1))

      log_warn "preflight cwd-map: skipping malformed line at $CWD_FROM:$cwd_lineno"
```
