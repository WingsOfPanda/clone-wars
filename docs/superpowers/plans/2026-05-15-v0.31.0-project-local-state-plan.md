# v0.31.0 Project-Local State Relocation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Relocate clone-wars state from global `~/.clone-wars/state/<hash>/<topic>/` to per-project `<invoking-cwd>/.clone-wars/state/<hash>/<topic>/`. Closes the cross-session `UserPromptSubmit` hook bleed and the args-file collision the user surfaced during the v0.30.0 implementation.

**Architecture:** Single canonical change in `lib/state.sh::cw_state_root` (returns `$PWD/.clone-wars` instead of `${CLONE_WARS_HOME:-$HOME/.clone-wars}`). `CLONE_WARS_HOME` env var kept as a test/debug seam (production never sets it; 96 tests use it for sandbox isolation). Hook patched to scan project-local root. Directives switch to `mktemp` for args files. `CW_TOPIC_REPO_CWD` env var fully removed from production code. `<sha256-of-cwd>` path segment kept for v0.31.0 — its removal deferred to v0.32.0 to avoid ~150 LoC of test churn unrelated to user-facing fixes.

**Tech Stack:** Pure bash 4.2+ + tmux 3.0+. No Node/Python runtime. Tests via `bash tests/run.sh`. Helpers: `tests/lib/assert.sh` (pass, assert_eq, assert_contains, assert_file_exists, assert_exit); inline `{ echo FAIL; exit 1; }` for failures. AskUserQuestion is invoked from Claude (slash directive), not bash — directive prose tests grep for the structural pattern.

```bash
# cw_state_root — absolute path to this project's clone-wars state root.
#
# Default: $PWD/.clone-wars (project-local; the directive's Bash blocks run
# in the conductor's invocation cwd, so $PWD resolves there).
#
# Test/debug seam: if CLONE_WARS_HOME is set, return its value verbatim.
# Production flow never sets it; 96 existing tests use the env var for
# sandbox isolation (they cd elsewhere but want state under a tmpdir).
#
# INVARIANT: never call inside a `cd` subshell — $PWD changes there. The
# only safe caller is the conductor's invocation cwd (which inherits $PWD
# from the slash directive's environment).
cw_state_root() {
  if [[ -n "${CLONE_WARS_HOME:-}" ]]; then
    printf '%s\n' "$CLONE_WARS_HOME"
  else
    printf '%s/.clone-wars\n' "$PWD"
  fi
}
```
