### Reconcile helper contract

`cw_deep_research_trooper_state_reconcile <art_dir> <commander>`:

- Reads `<art_dir>/troopers/<commander>/outbox.jsonl` starting at the byte offset stored in `<art_dir>/troopers/<commander>/liveness-cursor.txt` (or 0 if cursor missing/empty).
- Scans tail for the LAST `done` or `error` event (the LAST occurrence, not the first — if outbox tail has `[progress, done, progress, error]` the helper matches `error`).
- Resolves `current_exp_id` via `cw_deep_research_trooper_state_field "$art_dir" "$commander" current_exp_id`.
- Branches:
  - LAST event is `done` AND `<art_dir>/troopers/<commander>/experiments/<current_exp_id>/result.json` exists → `cw_deep_research_trooper_state_write "$art_dir" "$commander" phase=idle`.
  - LAST event is `done` AND result.json missing → leave state.txt unchanged.
  - LAST event is `error` → `cw_deep_research_trooper_state_write "$art_dir" "$commander" phase=failed`.
  - No terminal event in tail → leave state.txt unchanged.
- Idempotent — second call sees `phase=idle` or `phase=failed`, still scans, applies same branch, writes same value.
- Returns rc=0 on every branch; rc=1 only on missing `<art_dir>` or `<commander>` args.

### Forensics helper contract

`cw_spawn_capture_failure_forensics <commander> <model> <topic> <pane_id> <reason> [<event_line>]`:

- Resolves trooper dir via `cw_trooper_dir "$commander" "$model" "$topic"` (existing helper at lib/ipc.sh:13).
- Returns rc=1 if the dir is missing or unwritable (best-effort — spawn flow continues to archive even on rc=1).
- Validates `<reason>` ∈ {`timeout`, `error_event`} — anything else → rc=2.
- Writes `<trooper-dir>/failure-reason.txt` atomically via `cw_atomic_write`:

```
# Spawn bootstrap failure
timestamp:     <iso UTC, date -u +%Y-%m-%dT%H:%M:%SZ>
commander:     <commander>
model:         <model>
topic:         <topic>
pane_id:       <pane_id>
fail_reason:   timeout|error_event
ready_timeout: <CW_SPAWN_READY_TIMEOUT or "unknown">

## Pane scrollback (last 50 lines, captured BEFORE pane kill)
<tmux capture-pane -p -t "$PANE" | tail -n 50>

## Event context
<the {"event":"error",...} JSONL line if reason=error_event AND event_line passed;
 else "no error event before timeout">
```

- `ready_timeout` reads `$READY_TIMEOUT` from the caller's scope if set; else `"unknown"`.
- Returns rc=0 on successful write.

```bash
# cw_deep_research_trooper_state_reconcile <art_dir> <commander>
# v0.51 #3: pre-finalize state reconciler. Replays the trooper's outbox
# tail since liveness-cursor.txt and applies terminal done/error events
# to state.txt before finalize.sh's phase case-mapping runs. Catches the
# race where a done event arrives but the session ends before Yoda
# processes it — finalize would otherwise map phase=stale to
# phase=incomplete.
#
# Branches:
#   LAST event is `done` AND result.json exists → phase=idle
#   LAST event is `done` AND result.json missing → no write (incomplete is real)
#   LAST event is `error` (anywhere in tail) → phase=failed
#   No terminal event in tail → no write
#
# Idempotent — calling twice produces same final state.
# Returns rc=0 on every branch; rc=1 only on missing args.
cw_deep_research_trooper_state_reconcile() {
  local art_dir="${1:-}" cmdr="${2:-}"
  [[ -n "$art_dir" && -n "$cmdr" ]] || return 1
  local outbox="$art_dir/troopers/$cmdr/outbox.jsonl"
  local cursor_file="$art_dir/troopers/$cmdr/liveness-cursor.txt"
  [[ -f "$outbox" ]] || return 0
  local offset=0
  if [[ -f "$cursor_file" ]]; then
    offset=$(cat "$cursor_file" 2>/dev/null)
    [[ "$offset" =~ ^[0-9]+$ ]] || offset=0
  fi
  # Scan tail for terminal events; track LAST done and LAST error.
  local last_done="" last_error=""
  while IFS= read -r line; do
    case "$line" in
      *'"event":"done"'*)  last_done=1 ;;
      *'"event":"error"'*) last_error=1 ;;
    esac
  done < <(tail -c "+$((offset + 1))" "$outbox" 2>/dev/null)
  # error wins over done when both appear in the tail (matches the
  # "LAST terminal event" semantics — if an error follows a done, the
  # trooper is in a failed state).
  if [[ -n "$last_error" ]]; then
    cw_deep_research_trooper_state_write "$art_dir" "$cmdr" phase=failed
    return 0
  fi
  if [[ -n "$last_done" ]]; then
    local exp_id
    exp_id=$(cw_deep_research_trooper_state_field "$art_dir" "$cmdr" current_exp_id 2>/dev/null)
    if [[ -n "$exp_id" && -f "$art_dir/troopers/$cmdr/experiments/$exp_id/result.json" ]]; then
      cw_deep_research_trooper_state_write "$art_dir" "$cmdr" phase=idle
    fi
  fi
  return 0
}
```

Note on the "LAST event" semantics: the spec/T1-case-3 use "LAST event = error" to mean "if error appears anywhere in the tail it wins over done." This matches the trooper's actual failed state (an error after a premature done means things broke). The implementation uses a single-pass scan that tracks both `last_done` and `last_error`; if error is set at end-of-scan, error wins.

```
# cw_spawn_capture_failure_forensics <commander> <model> <topic> <pane_id> <reason> [<event_line>]
# v0.51 #5: capture spawn-bootstrap failure forensics to
# <trooper-dir>/failure-reason.txt BEFORE cw_state_archive moves the
# dir. Best-effort: returns rc=1 if the trooper dir is missing or
# unwritable so the spawn flow can still archive on failure.
#
# Fields written:
#   timestamp, commander, model, topic, pane_id, fail_reason, ready_timeout
#   ## Pane scrollback (last 50 lines from tmux capture-pane)
#   ## Event context (the JSONL line if reason=error_event AND event_line passed;
#                     else "no error event before timeout")

# Returns rc=0 on success, rc=1 on missing dir, rc=2 on invalid reason.
cw_spawn_capture_failure_forensics() {
  local commander="${1:-}" model="${2:-}" topic="${3:-}" pane_id="${4:-}" reason="${5:-}" event_line="${6:-}"
  [[ -n "$commander" && -n "$model" && -n "$topic" ]] || return 1

  local rt="${READY_TIMEOUT:-unknown}"
  local scrollback
  scrollback=$(tmux capture-pane -p -t "$pane_id" 2>/dev/null | tail -n 50 || true)
  local event_section
  if [[ "$reason" == "error_event" && -n "$event_line" ]]; then
    event_section="$event_line"
  else
    event_section="no error event before timeout"
  fi
  {
    printf '# Spawn bootstrap failure\n'
    printf 'timestamp:     %s\n' "$ts"
    printf 'commander:     %s\n' "$commander"
    printf 'model:         %s\n' "$model"
    printf 'topic:         %s\n' "$topic"
    printf 'pane_id:       %s\n' "$pane_id"

    printf 'ready_timeout: %s\n' "$rt"
    printf '\n## Pane scrollback (last 50 lines, captured BEFORE pane kill)\n'
    printf '%s\n' "$scrollback"
    printf '\n## Event context\n'
    printf '%s\n' "$event_section"
  } | cw_atomic_write "$out"
}
```
