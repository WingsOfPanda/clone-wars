**Goal:** Close eight quality-of-life gaps in `/clone-wars:deep-research` surfaced by the 2026-05-17 dogfood, grouped into five implementation lanes (A through E) + release plumbing (G).

**Architecture:** Each lane is one Conventional Commit, mostly independent (no shared file beyond `lib/deep-research.sh` touched by Lane D). The trooper-dispatch flow gains one new sentinel value (`phase=abandoned`) and one new optional pre-flight stage (skill-side smoke-test); both inert when not exercised. The archive flow gains two side effects (orphan sweep + winner symlink). The halt flow gains one unconditional render-summary call. The `halt.flag` write format standardizes across three writers.

**Tech Stack:** Pure bash 4.2+, tmux 3.0+, git ≥ 2.17. No Node/Python runtime. Tests via `tests/run.sh` (glob discovery of `test_*.sh`). Assert helpers at `tests/lib/assert.sh` (`pass`, `assert_eq`, `assert_contains`, `assert_exit`, `assert_file_exists`).
