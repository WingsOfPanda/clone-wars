# v0.29.0 simplification sweep — design

**Generated:** 2026-05-13
**Branch:** `chore/v0.29.0-simplification-sweep`
**Audit source:** code-simplifier sweep on 2026-05-13 (post-v0.28.3 strict-dogfood). Two agents reviewed the deep-research surface (~2,355 LoC) and the rest of the codebase (~79 files); together they surfaced 40 findings prioritized P0-P2.

## Goal

Remove dead code, dedup helpers, and clean stale doc drift across the clone-wars plugin in a single bundled v0.29.0 release. Estimated net delete: **~2,080 LoC**, no user-visible behavior change on any of the 5 slash commands (`/clone-wars:medic`, `/clone-wars:consult`, `/clone-wars:meditate`, `/clone-wars:deep-research`, `/clone-wars:deploy`).

## Architecture

Five thematic clusters land as separate commits on one branch, in this order:

```
A (deletion) → C (doc drift) → B (deep-research lib trim) → E (deep-research directive prose) → D (cross-command teardown/archive dedup)
```

Each cluster is independently revertable via `git revert <sha>` on `chore/v0.29.0-simplification-sweep`. The full test suite (`bash tests/run.sh`) must pass 0-FAIL after each cluster commit before moving to the next.

Out-of-scope (deferred per audit): `lib/consult.sh` 907-LoC file split (F-rest-021), wait-shim consolidation (F-rest-022/F-rest-009), `--args-file` boilerplate macro (F-rest-017), F-dr-013 render-function dedup, F-dr-014 `lib/_all.sh` source shortcut. These are flagged for a separate design pass.

## Failure modes

1. **Cluster A pre-flight grep finds a tracer reference we missed.** → Investigate; add the reference to the audit list; defer A until resolved.
2. **Cluster B1 grep finds a non-test plateau caller.** → Skip B1 entirely; remaining B sub-clusters proceed.
3. **Cluster D1 diff shows <80% overlap between archive scripts.** → Skip D1; remaining D sub-commit (D2) proceeds.
4. **A cluster commit breaks the test suite.** → `git reset --hard HEAD~1` on the branch; re-plan that cluster only; preceding clusters stay landed.
5. **Pre-existing flaky test fails on v0.29.0 baseline.** → Note in PR description; do not block.
